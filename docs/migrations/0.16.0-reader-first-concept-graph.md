# 0.16.0 reader-first concept graph migration

0.16.0 adds a rebuildable presentation renderer (`1.0.0`) and manifest `0.2.0`.
It does not alter immutable LearningCapture JSON, canonical publication plans, receipts, Approvals
or Proposals. The visible layout changes from `Data/Captures`, `Captures`, and `Views/Captures` to
reader-facing `学习记录` and `概念` Markdown files. The sync client removes an old managed file only
when its local digest still equals the recorded managed digest; local edits are preserved as conflicts.
