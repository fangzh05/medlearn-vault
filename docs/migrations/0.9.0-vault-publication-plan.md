# 0.9.0: deterministic Vault publication plan

Adds control-only `VaultPublicationPlan` 0.1.0 and the `medlearn workflow plan-publication` command.
It creates only immutable plan objects in `medlearn-control`; no Vault credential, access, writer,
sync manifest, Obsidian integration, or artifact upload is introduced. Existing persistent and
workflow contract versions are unchanged.
